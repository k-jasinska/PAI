# Projekt zaliczeniowy z przedmiotu: Projektowanie Aplikacji Internetowych

Aplikacja siłowni posiadająca dwa rodzaje użytkowników(zwykły, admin). 
W pełni responsywna.

### Użyte technologie:
  - PHP
  - HTML
  - CSS
  - AJAX
  - JS + jQuery
  - MariaDB
  - Bootstrap
